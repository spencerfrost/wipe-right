# Wipe Right

A toilet paper value calculator that helps you compare products and find the best deal.

## Features

- Compare multiple toilet paper products side-by-side
- Calculate price per 100 sheets and price per square foot
- Automatically highlights the best value
- Data persists in localStorage

## Getting Started

```bash
pnpm install
pnpm dev
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui (new-york style)
